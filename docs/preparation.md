## Objectif

Créer un site web moderne et responsive pour une agence de voyages fictive à Barcelone. Le site présente l’agence, ses services, des informations sur l’équipe, et permet de découvrir les offres principales de façon claire et professionnelle.

## Technologies utilisées

- React
- Next.js
- TailwindCSS
- GitHub pour le travail collaboratif
- Markdown pour la documentation

## Structure et pages du projet

- **Layout** :
  - Header (NavBar) avec navigation par ancres (Accueil, Nos services, À propos de nous)
  - Footer à 4 colonnes (Présentation, Services, Langues, Contact)
- **Page d’accueil** :
  - Banniere (image principale, titre, sous-titre)
  - Section Services (cartes avec images et descriptions)
  - Section Info (À propos de nous)

## Couleurs principales utilisées

- Jaune ambre clair : `#FEF3C7` (bg-amber-100)
- Noir : `#000000` (text-black)
- Bleu foncé pour certains titres : `#1D4ED8`
